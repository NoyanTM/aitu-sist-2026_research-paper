# aitu-sist-2026_research-paper
Research paper for [AITU SIST 2026 Conference](https://sist.astanait.edu.kz/)

## Description
- Based on "IEEE Conference Template" and AITU SIST requirements
- Conference track: "Computer science, IEEE Computer Society (CS), IEEE Transactions on Computers. Performance, fault tolerance, reliability, security, and testability"
- It is an draft whitepaper for `codecore` which initiates related ecosystem of solutions/methods (`codestash`, `fastsandbox`, `codeless`, `codeload`, etc.)

## Requirements
- Used software: drawio (.drawio), libreoffice (.odt), zotero (.bib), python (.py).
- /figures are built from /paper/diagrams.drawio.
- /paper contains latest attached paper and presentation.
- /practice contains a practical part that are related to discrete event simulation to represent how our preventive quality gate can help PyPI ecosystem in dynamics with additional web visualizations and gamifications.
